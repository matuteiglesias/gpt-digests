

# API Overview

The most useful modules:

- `digests_project.bags_pipeline.unitize` — core dataclasses and helpers for Units.
- `digests_project.bags_pipeline.ingest_logs` / `ingest_sessions` — normalize raw JSONL.
- `digests_project.bags_pipeline.pairs` — build pairbags.
- `digests_project.bags_pipeline.hydrate` — build indices and render hydrated Markdown.
- `digests_project.bags_pipeline.l2` — build & write L2 digests.
- `digests_project.bags_pipeline.select` — selectors for L3/L4/L5.
